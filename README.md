## ✨ Öne Çıkan Özellikler

* 🎓 **Güvenli Ağ:** Sadece `.edu.tr` uzantılı e-postalar ile kayıt olunabilir, dışarıdan kimse giremez.
* 🤝 **Kargo Derdi Yok:** Satıcı ve alıcı aynı kampüste! "Kargo bekleme, kantinde buluş" mantığıyla anında teslimat.
* 📚 **Öğrenciye Özel İlanlar:** Kitaptan mikroskoba, laboratuvar önlüğünden ders notlarına kadar kampüs hayatına dair her şey.
* ⚡ **Modern Altyapı:** Hızlı ve kesintisiz bir deneyim için güncel web teknolojileriyle geliştirildi.

---

## 📂 Proje Mimarisi (Monorepo)

Bu proje profesyonel bir monorepo yapısında kurgulanmıştır:
* 📁 `campusswap-api`: İş kurallarının, veritabanı işlemlerinin ve güvenlik katmanlarının bulunduğu **Backend** (NestJS) klasörü.
* 📁 `campusswap-frontend`: Kullanıcı arayüzünün ve modern tasarımların bulunduğu **Frontend** (React) klasörü.

---

## 🚀 Geliştirici Ortamı Kurulumu

Projeyi kendi bilgisayarınızda test etmek için aşağıdaki adımları sırasıyla uygulayabilirsiniz:

### 1. Backend'i (NestJS) Ayağa Kaldırma
`cd campusswap-api`
`npm install`
`npm run start:dev`

> API varsayılan olarak http://localhost:3000 portunda çalışacaktır.

### 2. Frontend'i (React) Ayağa Kaldırma
Yeni bir terminal sekmesi açarak:
`cd campusswap-frontend`
`npm install`
`npm run dev`

> Arayüze http://localhost:5173 adresinden ulaşabilirsiniz.
